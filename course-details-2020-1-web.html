Objetivos Gerais

O curso de desenvolvimento Web visa apresentar ao aluno as características de programação específicas para desenvolvimento de sistemas Web.

Objetivos Específicos

Por meio desse curso, o aluno poderá identificar a estrutura de um sistema Web, compreender o fluxo de execução desse tipo de sistema, bem como identificar linguagens utilizadas para o desenvolvimento dessas soluções

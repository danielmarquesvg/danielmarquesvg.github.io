Objetivos Gerais
Compreender, desenvolver e implementar testes de software, a partir da análise das regras de negócios de sistemas.


Objetivos Específicos
  • Apresentar as principais técnicas de teste;
  • Apresentar técnicas para geração de casos de teste;
  • Propiciar uma visão geral das principais ferramentas para automação de teste.
Unidade 1
  • Fundamentos em teste de software:
    • O que é teste e por que ele é necessário?
    • Termos e definições;
    • Consistência de um conjunto de casos de teste;
    • Qualidade e confiabilidade;
  • Papéis de teste de software


Unidade 2
  • Princípios e técnicas de testes de software
    • Teste de unidade
    • Teste de integração
    • Teste de regressão


Unidade 3
  • Testes não-funcionais
    • Escalabilidade
    • Segurança
    • Carga
    • Desempenho
  • Desenvolvimento orientado a teste
    • Conceitos de casos de teste de Segurança
    • Ciclo de vida x testes


Unidade 4
  • Testes alfa, beta e de aceitação
  • Ambientes de Teste e Ferramentas para Automação de Testes
  • Teste de aplicações para a Web

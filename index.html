CGEO

O CGEO é um Sistema de Apoio à Decisão para o mapeamento, análise e auxílio na resolução dos problemas urbanos da cidade de Campina Grande, sendo um protótipo inicial que servirá para auxiliar os gestores municipais. O sistema foi desenvolvido baseado no paradigma funcional utilizando a linguagem de programação PHP juntamente com as ferramentas de geolocalização e geocodificação do Google Maps Javascript API, possibilitando a visualização dos problemas dentro da visualização das ruas do Google Maps, como também fora através dos mapas.
